Host Discovery NSE Script

This is a simple NSE script that performs host discovery by sending an ICMP Echo Request packet to each target to determine whether it is live or not.

Script Information

Author DhaneshKumar
Categories Discovery
Description Performs a simple host discovery by sending an ICMP Echo Request packet to each target to determine whether it is live or not.

Usage

To use this script, simply run the following command:

nmap --script host-dis <target>

Replace <target> with the IP address or range of the target you want to scan.

Output

Host 192.168.254.16 is live.
Host 192.168.254.77 is live.

The script will output a list of live hosts found on the target network, along with their IP addresses and hostnames (if available).
